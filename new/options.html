Aide

Bienvenu dans ucbcChat, une application simple à utiliser

Creer un compte

Première étape, rassurez-vous que vous avez déjà ouvert un compte dans la bibliothèque virtuelle. Si cela est fait, vous utiliserez chaque fois votre mot de passe et votre numero matricule pour vous connecter.

Écrire un message

Les autres Rassurez vous d'abord que vous avec déjà choisi un correspondant. Cliquez alors sur l'icône de message pour écrire un message. Les messages échangées entre vous et lui seront vus sur cette page.

Voir les étudiants

Les autres Cliquez sur les autres, pour voir tous les édiants de l'UCBC et en cliquant sur le nom d'un des étudiants, vous pourez commencez à discuter avec lui en cliquant sur la petite icône de message.

Correspondants récents

Les autres Cliquez sur l'icône de l'horloge pour voir ceux avec qui vous avez communiqué, il n'y a pas longtemps.

Correspondants connectés

Les autres Cliquez sur l'icône de l'oeil pour voir les étudiants qui sont maintenant en ligne.

ucbcChat

ucbcChat est une application qui a été conçue en vue de permettre aux étudiants de l'Université Chrétienne Bilingue du Congo(UCBC) d'échanger des messages sur le campus.

Cette application devra être utilisée dans un but d'échanges qui respectent le manuel de l'étudiant.

Pour le moment, l'application est gratuite et peut donc être partagée.
